# 计算引擎

CUTE 的计算引擎由 MatrixTE（矩阵张量引擎）及其内部的 FPE（浮点运算引擎）组成，采用外积数据流和双流架构（Flow 1 标准多精度 + Flow 2 E2M1 高吞吐），支持从 INT8 到 FP32 以及 MXFP/NVFP 等 13 种数据类型。

## 模块总览

```
VectorA ──→ ┌─────────────────────────────────────┐
            │  MatrixTE (M×N PE Array)            │
VectorB ──→ │  每个 PE = FPE (Flow 1 + Flow 2)   │ ──→ MatrixD
ScaleA  ──→ │  D = Σ_k(scaleA × A × scaleB × B)+C│
ScaleB  ──→ └─────────────────────────────────────┘
MatrixC ──→
```

## FPE 四阶段逻辑架构

```
┌──────────────────────────────────────────────────┐
│                    FPE (FReducePE)               │
│                                                  │
│  Flow 1 (统一浮点路径)      Flow 2 (E2M1 定点)   │
│  ┌──────────────────────┐   ┌──────────────────┐ │
│  │ 所有格式统一处理     │   │ LUT → 定点乘法   │ │
│  │ 含 INT8/FP8/FP4     │   │ 绕过对阶逻辑      │ │
│  │ CmpTree + 27-bit对齐│   │ 2X / 4X 吞吐量   │ │
│  └──────────┬───────────┘   └────────┬─────────┘ │
│             │ 共享硬件资源 ↓          │           │
│  ┌──────────┴────────────────────────┴─────────┐ │
│  │  阶段 A: 预处理与解码 · 尾数乘法 · 指数预算 │ │
│  │  阶段 B: CmpTree · 27-bit 对齐阵列          │ │
│  │  阶段 C: 符号注入 · 多级归约树 · 溢出保护     │ │
│  │  阶段 D: CLZ · RNE 舍入 · 异常处理 · 打包    │ │
│  └──────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────┘
```
