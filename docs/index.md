# CUTE 设计文档


## 前言
本文档是 CUTE 硬件加速器的微架构设计文档，面向芯片设计工程师、验证工程师和系统集成工程师，提供以下内容：

- **可参数化配置的张量核设计**：每个模块的设计参数、功能描述、微架构设计和接口信号
- **多种数据格式支持**：支持的多种混合精度(BF16\MXFP\NVFP等)格式方案
- **软件编程接口**：自定义指令编码、异步编程模型
- **基本软件测试**: 包含基本的验证测试方案介绍

本文档与 CUTE 源代码仓库保持同步，所有设计描述均基于源码实现，并通过论文（DAC 2026）进行交叉验证。
