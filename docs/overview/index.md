# 项目概述

CUTE（**C**PU-centric and **U**ltra-utilized **T**ensor **E**ngine）是一个面向 RISC-V 处理器的可配置矩阵加速器，以 RoCC 协处理器形式集成在 CPU 核内部，通过 TileLink 总线访问主存。
